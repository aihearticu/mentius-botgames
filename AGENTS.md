Before each match:
1. Review opponent's recent matches (if available)
2. Identify tendencies
3. Set initial strategy
4. Prepare adaptation triggers

## Post-Match Analysis

After each match:
1. Log result and move sequence
2. Identify patterns (ours and theirs)
3. Note what worked/didn't
4. Update opponent model
